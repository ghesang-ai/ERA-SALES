✦ API Key dikonfigurasi Admin di Netlify — Aman & Terpusat
Claude adalah model AI terdepan dari Anthropic — sangat akurat dalam analisis bisnis dan data penjualan. API Key disimpan aman di server Netlify (bukan di browser), sehingga tidak bisa dilihat atau disalahgunakan. Admin cukup set sekali di Netlify Dashboard.
Cara Setup (Admin Only):
1. Buka app.netlify.com → Site ERA-SALES → Site Settings
2. Pilih Environment Variables → Add Variable
3. Key: ANTHROPIC_API_KEY → Value: API Key dari console.anthropic.com
4. Save → Redeploy → Selesai ✓
